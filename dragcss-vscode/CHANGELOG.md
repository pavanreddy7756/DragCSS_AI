# Changelog

## 1.0.0

- WebSocket server for Chrome extension connection
- CSS file finder (searches .css, .scss, .less, .sass, .styl files)
- Auto-apply CSS property changes to matched rules
- Override file generation when no matching rules found
- Copilot Chat Participant (@dragcss) with /apply, /show, /status commands
- Status bar indicator showing connection state
- Configurable port, auto-apply, and auto-start settings
