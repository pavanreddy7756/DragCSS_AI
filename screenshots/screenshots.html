📸 How to take screenshots for Chrome Web Store

  2. For each slide below, take a screenshot of just the slide area (Cmd+Shift+4, then drag over the slide)
  3. Each slide is exactly 1280×800px — the required Chrome Web Store screenshot size
  4. Upload all 5 screenshots in order to the Chrome Web Store listing
